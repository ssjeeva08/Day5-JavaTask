# Java Collections Framework Programs

This repository contains Java programs that demonstrate the usage of the Java Collections Framework and its commonly used classes.

## Topics Covered

- ArrayList
- LinkedList
- Vector
- HashSet
- LinkedHashSet
- TreeSet
- Comparable Interface
- Collection Operations
- Sorting and Ordering
- Duplicate Handling

## Purpose

These exercises help in understanding different collection types, their features, and how data is stored, managed, and retrieved in Java.
